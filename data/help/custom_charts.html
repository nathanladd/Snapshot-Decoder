Custom Charts

Custom charts allow you to create your own parameter combinations and visualizations beyond the pre-configured Quick Charts. This gives you complete flexibility to analyze any combination of PIDs from your snapshot data.

Creating a Custom Chart

To create a custom chart, you'll use the PID Descriptions window to select and add parameters to your chart axes.

Step 1: Open the PID Descriptions Window

Access the PID Descriptions window from the View menu. This window displays all available parameters from your loaded snapshot.

See PID Descriptions Window for detailed information.

Step 2: Search for PIDs

Use the search box at the top of the PID Descriptions window to quickly find parameters:

  • Type any part of a PID description (e.g., "pressure", "temp", "fuel")
  • The list automatically filters to show only matching PIDs
  • Search is not case-sensitive
  • Clear the search to see all PIDs again

Step 3: Add PIDs to Chart Axes

Once you've found the parameters you want to chart:

  1. Double-click a PID in the list to open the add-to-chart dialog
  2. Choose Add to Primary Axis or Add to Secondary Axis
  3. The PID is immediately plotted on the selected axis
  4. Repeat for additional parameters
Tip: Use the primary axis for your main parameters of interest, and the secondary axis for parameters with different units or scales that you want to compare.

Modifying Axis Ranges

After adding PIDs to your chart, you can adjust the Y-axis ranges to focus on specific value ranges or improve readability.

Using Axis Range Controls

The main window provides controls for adjusting axis ranges:

  • Locate the axis range controls in the main window
  • Enter minimum and maximum values for the primary (left) Y-axis
  • Enter minimum and maximum values for the secondary (right) Y-axis if used
  • The chart updates automatically to reflect your changes

Adjusting axis ranges helps you:

  • Zoom in on specific value ranges of interest
  • Remove outliers or noise from the view
  • Align multiple parameters for easier comparison
  • Create consistent scales across multiple charts

Exporting and Saving Charts

Once you've created a custom chart, you have two options for saving your work:

Save Individual Chart as PDF

Use the Save button in the chart toolbar to export a single chart:

  • Click the Save button (floppy disk icon) in the chart toolbar
  • The chart is saved as a single-page PDF file
  • Snapshot metadata (serial number, engine hours, date) is automatically embedded in the PDF
  • Choose a location and filename for your PDF

See Chart Toolbar for more information about toolbar functions.

Add Multiple Charts to Cart

Use the Add to Cart button to collect multiple charts for batch export:

  • Click the Add to Cart button (shopping cart icon) in the chart toolbar
  • The current chart configuration is added to your cart
  • Create additional custom charts and add them to the cart as well
  • Access the chart cart to review all saved chart configurations
  • Export all charts at once or manage individual cart items
Workflow Tip: Use the chart cart when you need to create a set of diagnostic charts that you want to review or export as a single pdf file. This is especially useful for creating consistent diagnostic reports that you can attach to your repair order.

Tips for Effective Custom Charts

  • Group related parameters - Put similar measurements on the same axis for easier comparison
  • Use descriptive searches - Search for terms like "fuel", "air", "temp" to find related PIDs quickly
  • Adjust ranges strategically - Set axis ranges to highlight the data that matters most
  • Save frequently used combinations - Add common chart configurations to the cart for quick access
  • Start with Quick Charts - Use Quick Charts for initial diagnostics, then create custom charts to investigate specific issues

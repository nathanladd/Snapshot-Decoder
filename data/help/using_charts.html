Using Charts

Snapshot Decoder provides powerful charting capabilities to visualize diagnostic data over time.

Quick Charts

Quick charts are pre-configured visualizations designed for common diagnostic scenarios. After loading a snapshot, relevant quick chart buttons appear based on the snapshot type.

Simply click a quick chart button to instantly generate the visualization. Examples include:

  • Battery Test - Battery voltage vs RPM
  • Rail Pressure - Demand vs actual rail pressure
  • Turbo - Manifold and atmospheric pressure with RPM
  • EGR Position - EGR position vs demand percentage

Custom Charts

Create custom charts by selecting PIDs from the dropdown menus:

  1. Select a primary PID from the left dropdown
  2. Optionally select additional PIDs for comparison
  3. Click Chart → Plot Selected PIDs or use the plot button

Chart Controls

Toolbar

ToolFunction
HomeReset to original view
PanLeft-Click and drag to move the view. Right-Click and drag to zoom
ZoomDraw a rectangle to zoom into an area
SaveExport the chart as a pdf

Cursor Options

Enable cursors from the Chart menu:

  • Show Cursor - Displays a vertical line you can drag across the chart
  • Show Values - Shows exact values when hovering over data points
Tip: Use Chart → Pop Out Chart to open the current chart in a separate resizable window for detailed analysis or presentations.

Chart Cart

Place multiple charts in the cart to be saved a single, multi-page pdf:

  1. Create a custom chart with your desired PIDs
  2. Click Chart → Add to Cart
  3. Charts added to the cart are displayed in the Chart Cart panel on the right side
Note: Chart cart items are not saved between sessions.

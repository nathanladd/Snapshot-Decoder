Getting Started

This guide will help you get up and running with Snapshot Decoder quickly.

Basic Workflow

Step 1: Open a Snapshot File

Click File â†’ Open Snapshot or press Ctrl+O to open a snapshot file. Supported formats include .xls and .xlsx files exported from Bobcat Engine Analyzer.

Step 2: Select a Quick Chart

Once loaded, the application detects the snapshot type and shows relevant quick chart buttons. Click any button to instantly visualize that diagnostic data.

Step 3: Analyze the Data

Use the interactive chart features to zoom, pan, and inspect specific data points. Enable cursors to read exact values at any point in time.

Step 3: Export charts as .pdf

Use the save command in the chart commmand bar to export individual charts. Add multiple charts to the Chart Cart and export them all to one pdf.

System Requirements

  • Windows 10 or later
  • Screen resolution of 1280x720 or higher recommended
  • Complete snapshot files in .xls or .xlsx format

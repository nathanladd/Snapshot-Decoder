Keyboard Shortcuts

Use these keyboard shortcuts to work more efficiently with Snapshot Decoder.

File Operations

ShortcutAction
Ctrl + OOpen snapshot file
Ctrl + WClose current snapshot
Alt + F4Exit application

Chart Navigation

ShortcutAction
HomeReset chart view
Left ArrowMove cursor left
Right ArrowMove cursor right
Mouse WheelZoom in/out

View Options

ShortcutAction
F1Open Help
F11Toggle fullscreen

Mouse Actions

ActionFunction
Click + DragPan chart (when pan tool active)
Click + Drag RectangleZoom to area (when zoom tool active)
Double ClickReset zoom to fit all data
Right ClickContext menu (where available)

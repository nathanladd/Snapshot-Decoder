Chart Controls

Snapshot Decoder provides comprehensive chart controls that allow you to customize and interact with your diagnostic charts. The chart controls are divided into two main areas: the axis controls in the left panel and the toolbar at the top of each chart.

Overview

The chart controls give you complete flexibility to:

  • Select which PIDs to display on primary and secondary axes
  • Arrange the display order of PIDs
  • Set custom axis ranges for precise analysis
  • Navigate, zoom, and interact with chart data
  • Export charts for documentation and reporting

Live Chart Updates

Most chart controls have a live effect on the current chart, meaning changes are applied immediately without requiring additional action:

Controls with Live Updates

  • Adding PIDs - Charts update automatically when you add PIDs to either axis
  • Removing PIDs - Deleting a PID from an axis immediately removes it from the chart
  • Reordering PIDs - Moving PIDs up or down with the â–² and â–¼ buttons instantly changes the display order and z-order
  • Toolbar interactions - Zoom, pan, cursor, and values tools activate immediately when clicked

Axis Range Changes Require Plot Chart

Important: When you change the min or max values for axis ranges, you must click the Plot Chart button to apply the changes. This is the only chart control that does not update live.

This design allows you to adjust both primary and secondary axis ranges together before regenerating the chart, avoiding multiple unnecessary redraws.

Typical Workflow

Here's how the chart controls work together in a typical analysis session:

  1. Select PIDs - Search and add PIDs to primary or secondary axes (live update)
  2. Arrange display - Use â–² and â–¼ buttons to order PIDs (live update)
  3. Set axis ranges - Optionally uncheck auto scaling and enter min/max values
  4. Click Plot Chart - Apply axis range changes if you made any
  5. Interact with chart - Use toolbar to zoom, pan, and measure values (live)
  6. Export or save - Add to cart or save as PDF using toolbar buttons (live)

Tips for Effective Chart Control

  • Start with auto scaling - Let the chart automatically determine the best axis ranges, then adjust manually if needed
  • Use consistent ranges - When comparing multiple snapshots, set the same axis ranges to make visual comparison easier
  • Reorder for visibility - If important data is hidden behind other lines, move it to the bottom of the list to draw on top
  • Combine axes strategically - Group PIDs with similar ranges on the same axis for clearer visualization
  • Remember to click Plot Chart - After changing axis min/max values, don't forget to click Plot Chart to see the changes
Quick Tip: You can quickly reset your chart by clicking the Clear Chart button, which removes all PIDs from both axes and returns the controls to their default state.

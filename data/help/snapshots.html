Working with Snapshots

Snapshot Decoder supports multiple snapshot formats from different ECU generations and diagnostic tools.

Supported Snapshot Types

ECU V1 Snapshots

First-generation ECU snapshots with comprehensive engine diagnostic data. These include PIDs for:

  • Engine speed and load
  • Fuel system (rail pressure, IMV current)
  • Air system (turbo, EGR, mass air flow)
  • Electrical (battery, sensors)

ECU V2 Snapshots

Second-generation ECU snapshots with extended diagnostic capabilities and additional parameters.

ECU2 Snapshots

Latest generation snapshots with the most comprehensive data set available.

Auto-Detection: Snapshot Decoder automatically detects the snapshot type when you open a file and configures the available quick charts accordingly.

File Formats

The following file formats are supported:

  • .xlsx - Excel workbook (recommended)
  • .csv - Comma-separated values

Opening Snapshots

  1. Click File → Open Snapshot or press Ctrl+O
  2. Navigate to your snapshot file
  3. Select the file and click Open
  4. The application will load and analyze the data

Once loaded, the header panel shows key information about the snapshot including:

  • Equipment serial number
  • Date and time of capture
  • Snapshot type detected
  • Number of data points

Viewing Raw Data

To see the underlying data:

  • Raw Data - Data → Raw Data shows the original imported data
  • Clean Table - Data → Clean Table shows processed data with units
  • PID Descriptions - Data → PID Descriptions explains each parameter

First Steps with Snapshot Decoder

This guide walks you through a typical workflow for analyzing a snapshot file, from opening the file to creating a diagnostic report. Follow these steps to get started quickly.

Step 1: Open a Snapshot File

Begin by loading your snapshot file into Snapshot Decoder:

  1. Go to File â†’ Open Snapshot in the menu
  2. Browse to your snapshot file (.xls or .xlsx format)
  3. Select the file and click Open

Snapshot Decoder will automatically:

  • Detect the snapshot type (V1 ECU, V1 EUD, or V2 ECU)
  • Load and parse the raw snapshot data
  • Display header information including engine hours, serial number, and snapshot date
  • Configure the available Quick Charts for your snapshot type

See Opening Files for more details.

Step 2: Review Snapshot Header Information

After loading the snapshot, review the header information displayed at the top of the main window. This shows important details about the engine and when the snapshot was captured.

See Snapshot Header for information about the displayed fields.

Step 3: Open Quick Charts

Quick Charts provide instant access to the most important diagnostic views. Start your analysis by reviewing relevant Quick Charts:

Recommended Starting Points

  • Battery Voltage - Check electrical system health during cranking and running
  • Rail Pressure - Verify fuel system pressure control and performance
  • Engine Use Data - Review operating conditions and history

To open a Quick Chart:

  1. Locate the Quick Charts panel at the top of the main window
  2. Click on any Quick Chart button
  3. The chart opens immediately with pre-configured parameters

See Quick Charts for more information about available charts.

Step 4: Analyze Chart Data

Use the chart toolbar to interact with and analyze your data:

  • Zoom - Click and drag to zoom into specific areas of interest
  • Pan - Move around the chart to view different sections
  • Vertical Cursor - Align readings across multiple parameters
  • Values - View exact data values at any point

See Chart Toolbar for detailed information about chart controls.

Step 5: Add Charts to Cart

As you review different Quick Charts, add the important ones to your Chart Cart for later export:

  1. With a chart open, click the Add to Cart button (shopping cart icon) in the chart toolbar
  2. The chart configuration is saved to your cart
  3. Open additional Quick Charts and add them to the cart as well
  4. Repeat for all charts you want to include in your diagnostic report
Tip: Add charts to the cart in the order you want them to appear in your final PDF report. This creates a logical flow for reviewing the diagnostic data.

Step 6: Export Charts as PDF

Once you've added all relevant charts to the cart, export them as a single PDF file:

  1. Access the Chart Cart from the main window
  2. Review the charts in your cart to ensure you have everything needed
  3. Click the Export or Save All button
  4. Choose a location and filename for your PDF report
  5. The PDF is created with all charts and embedded snapshot metadata

The exported PDF includes:

  • All charts from your cart in a single document
  • Snapshot metadata (engine serial number, engine hours, snapshot date) on each page
  • Professional formatting suitable for diagnostic reports
Workflow Tip: This PDF can be attached to repair orders, shared with colleagues, or kept as documentation of the diagnostic work performed. The embedded metadata ensures traceability back to the original snapshot.
